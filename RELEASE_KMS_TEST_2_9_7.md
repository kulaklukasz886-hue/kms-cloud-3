# KMS TESTOWY 2.9.11 — KOMPLETNY ETAP 2 DO TESTÓW

Ta wersja wdraża pełny model elementu produkcyjnego w Etapie 2.

Każdy element zawiera: INFO, nazwę, typ produktu, producenta, symbol katalogowy, format produktu, wymiary, ilość, grubość, OBROTOWO, kierunek dekoru, cztery krawędzie, technologię, wiele operacji CNC/frezowania, uwagi oraz status weryfikacji.

Nowe technologie: BLAT 38MM (gotowy produkt bez klejenia) i HPL 12MM.

Wersja jest przeznaczona do dalszych testów i wskazywania braków przez Łukasza. Parametry konkretnych wierceń i frezów nie są zgadywane — będą przypisywane z rzeczywistych programów CNC.
